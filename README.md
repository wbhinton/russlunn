# Russ Lunn for Macon County Commissioner

Official website for Russ Lunn's campaign for Macon County Commissioner, NC. Built with a focus on high performance, regional aesthetics, and transparency.

## 🚀 Tech Stack

- **Framework**: [Astro 5](https://astro.build/) (Content Layer)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons**: [Lucide Astro](https://lucide.dev/)
- **Typography**: Montserrat (Headings) & Inter (Body)

## 🎨 Design Theme

The site reflects the "Western NC" mountain community through a carefully curated palette:
- **Primary (Mountain Green)**: `#2D5A27`
- **Secondary (Unity Blue)**: `#0044CC`
- **Background (Soft White)**: `#F8FAFC`

## 📂 Project Structure

- `src/content/`: Managed content collections for **Issues** and **Press Releases**.
- `src/components/`: Reusable UI components (Donate Button, Issue Cards, Layout, etc.).
- `src/layouts/`: Base layout with built-in SEO and OpenGraph metadata.

## 🛠️ Commands

All commands are run from the root of the project:

| Command | Action |
| :--- | :--- |
| `npm install` | Installs dependencies |
| `npm run dev` | Starts local dev server at `localhost:4321` |
| `npm run build` | Build the production site to `./dist/` |
| `npm run preview` | Preview your build locally |

## 🛰️ Deployment

- **Development**: Automatically deployed to GitHub Pages via GitHub Actions on every push to `main`.
- **URL**: `https://wbhinton.github.io/russlunn/`
- **Production**: Planned for Netlify.
